# Frontend

## :computer: Run it!

```bash

# Clone the project

git clone https://github.com/felipe-jm/be-the-hero.git

cd be-the-hero

# Get into the backend

cd backend

# Install the dependencies

npm install
# or
yarn install

# Run the migrations

knex migrate:latest

# Run the backend

npm start
# or
yarn start 

# Get into the fronted

cd frontend

# Install the dependencies

npm install
# or
yarn install

# Start the project

npm start
# or
yarn start

# Get into the mobile

cd mobile

# Install the dependencies

npm install
# or
yarn install

# Start the project

npm start
# or
yarn start

```

Then you just need to have the _:point_right: <a href="https://play.google.com/store/apps/details?id=host.exp.exponent&hl=pt_BR">Expo app</a> :point_left:_ installed in your phone and scan the qrcode provided :ok_hand:
